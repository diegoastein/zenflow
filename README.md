# zenflow
generador de ruido blanco
🌊 ZenFlow - Ruido Blanco & Enfoque

ZenFlow es una aplicación web minimalista diseñada para mejorar la concentración, reducir el estrés y ayudar a la productividad mediante el uso de paisajes sonoros y ruidos de frecuencia constante.

🚀 Características

Audio Híbrido: Combina ruidos sintéticos (Blanco y Marrón) generados matemáticamente con muestras reales de alta calidad (Lluvia y Bosque).

Temporizador Integrado: Basado en bloques de tiempo (25m/50m) para técnicas de enfoque como Pomodoro.

Diseño Glassmorphism: Una interfaz moderna, oscura y elegante que no distrae al usuario.

Optimizado para Publicidad: Espacios integrados para anuncios nativos y banners sin interrumpir la experiencia de usuario.

🛠️ Tecnologías

HTML5 & CSS3 (Tailwind CSS para el estilizado).

Web Audio API: Para la generación de ruidos en tiempo real y el control de nodos de ganancia.

JavaScript Moderno: Gestión de estados de audio, temporizadores y manipulación del DOM.

📦 Instalación y Uso

Al ser una aplicación de un solo archivo (index.html), no requiere instalación:

Clona este repositorio.

Abre index.html en cualquier navegador moderno.

¡Empieza a enfocarte!
